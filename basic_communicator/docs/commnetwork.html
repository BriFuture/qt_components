    Detailed Description

    \authorBriFuture

    网络连接方式，接收服务器数据或将数据发送到相应的服务器端口中

    与其他连接方式不同，该方式使用 TcpSocket 连接到远程服务器中（而不是常用的串口）， 但由于 TcpSocket 继承自 QIODevice，因此实现方式与串口的实现类大同小异
